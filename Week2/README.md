# Week 2 OOP Calculator Project

## 📌 Overview
This project is an Object-Oriented Programming (OOP) implementation of a basic calculator developed as part of Week 2 Activity 3.

The program performs basic mathematical operations such as:
- Addition
- Subtraction
- Multiplication
- Division

## 🧠 OOP Concepts Used
- **Class:** Calculator
- **Methods:** add(), subtract(), multiply(), divide()
- **Encapsulation:** All operations are grouped inside a class

## ⚙️ Functional Design
The project also uses two functions outside the class:
- get_numbers(): Collects user input
- run_calculator(): Controls program flow and user interaction

## 🚀 How to Run
```bash
python basic_calculator.py
