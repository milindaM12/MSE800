# Student Activity Logger using Python Decorators

## Overview

This project demonstrates how Python decorators can be used to add reusable logging functionality to multiple functions without modifying their original implementations.

The project simulates a simple student activity system where activities such as:

- Student login
- Assignment submission
- Viewing grades

are automatically logged using a custom decorator.
