# Zoo Application Login System

## Description
This is a simple Python console-based Zoo Application Login System created using Object-Oriented Programming (OOP).

The project includes an admin login system and demonstrates the use of a Python decorator to protect certain functions from unauthorized access.

---

## Features

- Admin login authentication
- View zoo animals
- Add new animals
- Logout functionality
- Decorator-based access control
